函数有界的证明

Posted on 2023-12-17

Definition

设函数f(x)的定义域为D，对于任意的x∈D，均能找到一个大于0的M使得|f(x)|<=M，就称函数有界，如果这样的M不存在，就称函数无界

证明：

证明思路：

  1. 根据函数有界的定义，对于任意的M>0，均有|f(x)|<=M，只需找到一个M使得不等式不成立即可

  2. 因为x是变化的，所以我们要找到一个带x的表达式来表示M，让不等式成立或不成立，才能说明对于定义域内的任意的x，M都大于(即无界)或小于等于(即有界)函数值。

  3. 如果我们找到一个M不是用x来表示的而是一个固定的数值如一个很小的数值如0.0000001（或者很大的数值如1000000），那么我们是不能100%确定不等式永远成立的。

    (比如现在我们要证明函数无界即期望|f(x)|>M总是成立，根据直觉我们要找一个很大的M来证明不管M多么大不等式均成立，不妨设M为一千万，那么根据M的设定，我只需要让x=0.1(或者任意的零点几)，就能得出 即|f(x)|>M不成立了，所以我们无法证明或者说无法100%证明函数无界。这是为什么要用x来表示M的原因)

  4. 现在我们用x来表示M，因为函数有界的定义|f(x)|<=M，，在这道题中是 ｜1/x｜<= M，那让M = x 不行？或者等于x+1行不行

  5. 将上面的M带入不等式后得到，因为x ∈(0,1)，所以显然这个不等式不成立，也就是证明函数有界失败，即函数无界

  1. 使用反证法，证明函数有界
  2. 根据函数有界的定义，对于任意的M>0，均有
  3. 那让M=x，或者m=x+1
  4. 带入到不等式中得
  5. 因为x∈ 开区间0-1，所以显然不等式不成立，即函数有界不成立，即函数无界

  2. 让M=2（或任意大于1的数量）

Definition

如果有函数f(x)，当x趋于X时，极限为0，则称f(x)为x->X时候的无穷小。

注意点：

1，x趋于X时的极限为0

2，f(x)为无穷小，即无穷小是一个函数非一个固定的数

3，0是一个无穷小，无穷小不是0

Theory

2，A+å的意思是函数可以写成A这个常量加上一个函数的过程，这个函数是一个无穷小，而且变化过程和函数一样

充分与必要条件的区别

Posted on 2023-12-16

充分条件

若A是B的充分条件，则A与其他B的充分条件均可单独推出B，各充分条件之间属于平级并列关系，

反之B成立无法100%推出它的某一充分条件成立，图示如下：

必要条件

若A是B的必要条件，则A与B的其他必要条件都成立时才可推出B，A+C+D->B，

反之B成立可推出它的任意必要条件成立，图示如下：

证明

证明A是B的充分必要条件时，通常按照下面两部来进行

1，先证明充分性，即A能推出B

因为已知A，根据xxx，证得B，所以A是B的充分条件

2，再证明必要性，即B能反推出A

已知B成立，根据xxx，证得A，所以A是B的必要条件

函数的极限

定义

设一函数f(x)在某一去心邻域U(x0)内有定义，对于任意的ε大于0，存在一个δ大于0，当x与x0的距离大于0小于δ 时，有f(x)与一个常数A的距离小于ε 。则称A为函数f(x)在x趋于x0时的极限。记为

用数学语言表示

例

a对于常数函数f(x)=c，证明x趋于x0时的极限为c

证：

1，根据函数极限的定义，任取一个ε大于0，存在一个δ大于0，当0<｜x-x0｜<δ时，|fx-c|<ε。

2，因为 fx-c = 0, 0 < ε，

3，又因ε是任意的一个大于0的数，与δ没有约束关系，我们可任意取δ都能使得不等式成立

4，所以极限为c成立

b证明

1，根据函数极限定义，任取一个大于0的ε，存在一个大于0的δ，当xxx时，|f(x)-x0|<ε成立

2，化简不等式，得到x0-x0=0 < ε

3，所以函数本身对于x没有任何约束，所以就对δ没有约束，所以δ可以任意取

c证明

1，根据函数极限定义，对任意大于0的ε，都有大于0的δ，当0<|x-1|<δ，|2x-1-1|<ε成立

2，化简得：｜2x-2｜<ε 、|x-1| < ε /2

3，第2步的不等式就是函数极限定义对fx的约束，fx又对自变量x的约束，

4，根据得到的约束看能否找到一个δ，因为δ是要大于|x-1|的，而在函数极限定义对函数的约束中知道|x-1| < ε /2

5，那让δ等于 ε /2这个数字，即不论给的ε是多少，只要找到 ε /2作为δ，|fx-A|<ε 都能成立

理解

1，对于函数极限的证明，通常是根据函数极限定义，在已知题目中的极限值情况下，对｜fx-极限值｜< ε，这个不等式进行化简

2，化简后能够得到一个关于x的不等式，这个不等式是函数满足极限定义情况下，函数本身对于x的约束

3，根据函数对x的约束，我们要寻找一个δ，让δ满足 0<|x-x0|<δ，同时其中的x也要满足函数对x的约束

数列的极限

Posted on 2023-12-15 Edited on 2023-12-16

第一章 第二节 数列的极限

定义

设{Xn}为一个数列，如果存在一个常数a，对于任意给定的一个无论多么小的数ε>0，总存在一个正整数N，使得|{Xn}-a|<ε。则称a为数列的极限，或者称数列收敛于a。如果不存在这样的数，则称数列不存在极限。
$$
有数列a_n，一个常数a，\forall\epsilon>0，\exist N>0，当n>N时，|a_n-a|<\epsilon .
$$

注意：在利用数列极限的定义来论证某个数a 是数列的极限时，重要的是对于任意给定的正数ε ，要能够指出定义中所说的这种正整数 N确实存在

例子

$$
证明a_n = \frac{n + (-1)^{n-1}}{n}的极限为1.\
证：1，根据函数定义，有|a_n-1|<\epsilon，将a_n化简后得|\frac{1}{n}|<\epsilon，即\frac{1}{n}<\epsilon。即n>\frac{1}{\epsilon}\
2，那我们取N=\frac{1}{\epsilon}，就能满足定义中的(总能找到一个N>0，当n>N时。。。)这句话
$$

性质

  1. 极限的唯一性，如果数列a_n收敛，那么它的极限唯一

    用反证法，即数列 an 的极限即等于a又等于b，且a<b，然后取ε为 (b-a)/2，带入到两个极限式子中去
