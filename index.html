函数的极限

定义

设一函数f(x)在某一去心邻域U(x0)内有定义，对于任意的ε大于0，存在一个δ大于0，当x与x0的距离大于0小于δ 时，有f(x)与一个常数A的距离小于ε 。则称A为函数f(x)在x趋于x0时的极限。记为

用数学语言表示

例

a对于常数函数f(x)=c，证明x趋于x0时的极限为c

证：

1，根据函数极限的定义，任取一个ε大于0，存在一个δ大于0，当0<｜x-x0｜<δ时，|fx-c|<ε。

2，因为 fx-c = 0, 0 < ε，

3，又因ε是任意的一个大于0的数，与δ没有约束关系，我们可任意取δ都能使得不等式成立

4，所以极限为c成立

b证明$\lim_{x\to x0}f(x)=x0$

1，根据函数极限定义，任取一个大于0的ε，存在一个大于0的δ，当xxx时，|f(x)-x0|<ε成立

2，化简不等式，得到x0-x0=0 < ε

3，所以函数本身对于x没有任何约束，所以就对δ没有约束，所以δ可以任意取

c证明$\lim_{x \to 1}(2x-1)=1$

1，根据函数极限定义，对任意大于0的ε，都有大于0的δ，当0<|x-1|<δ，|2x-1-1|<ε成立

2，化简得：｜2x-2｜<ε 、|x-1| < ε /2

3，第2步的不等式就是函数极限定义对fx的约束，fx又对自变量x的约束，

4，根据得到的约束看能否找到一个δ，因为δ是要大于|x-1|的，而在函数极限定义对函数的约束中知道|x-1| < ε /2

5，那让δ等于 ε /2这个数字，即不论给的ε是多少，只要找到 ε /2作为δ，|fx-A|<ε 都能成立

理解

1，对于函数极限的证明，通常是根据函数极限定义，在已知题目中的极限值情况下，对｜fx-极限值｜< ε，这个不等式进行化简

2，化简后能够得到一个关于x的不等式，这个不等式是函数满足极限定义情况下，函数本身对于x的约束

3，根据函数对x的约束，我们要寻找一个δ，让δ满足 0<|x-x0|<δ，同时其中的x也要满足函数对x的约束

比如$\lim_{n \to 1} f(x)=\frac{n^2-1}{n-1}=2$(不太符合直觉…)

第一章 第二节 数列的极限

定义

设{Xn}为一个数列，如果存在一个常数a，对于任意给定的一个无论多么小的数ε>0，总存在一个正整数N，使得|{Xn}-a|<ε。则称a为数列的极限，或者称数列收敛于a。如果不存在这样的数，则称数列不存在极限。

注意：在利用数列极限的定义来论证某个数a 是数列的极限时，重要的是对于任意给定的正数ε ，要能够指出定义中所说的这种正整数 N确实存在
