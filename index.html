无穷小相关的定理及定义

定义

有函数f(x)，当函数极限为0时，f(x)成为0趋于x0（或者0小于N）时候的无穷小。

  1. 无穷小是一个函数，非数值
  2. 0是一个特殊的无穷小

定理：与函数极限的关系

在自变量的同一变化过程x趋于x0或者x趋于无穷时，函数f(x)具有极限的充分必要条件是f(x)=A+å。

其中A是函数极限，å是无穷小

  1. 前提条件：f(x)=A+å，其中A是一个常数、å是一个无穷小
  2. 结论：那么其中的f(x)极限一定存在且为A

定理1：两个无穷小的和/差是无穷小

证明：

前提：有无穷小a和b，都是x趋于x0时候的无穷小。

设：

有 C = a + b

因a和b为无穷小，则有对于任意的ε>0,都存在δ>0，当0<|x-x0|<δ1时，满足|a|<ε；当0<|x-x0|<δ2时，满足|b|<ε；

我们让δ = min{δ1，δ2}，则δ是能同时满足a和b的一个δ，将a和b相加得到|a|+|b| = ε+ε①。

现我们对C求极限：lim C = lim (a + b)，即对于任意的ε>0，是否能找到一个δ>0，当0<|x-x0|<δ时，|C| <ε？

可以的，因为 |C| = ｜a+b｜，而且因为｜a+b｜<= |a| + |b|，又因为|a|+|b| = ε+ε①，所以｜a+b｜<2ε，(ε和常数相乘对结果无影响)，所以C是无穷小

函数极限的运算法则

基于上面无穷小的相关定理，可以简化函数极限的运算

定理：函数的极限的四则运算法则

如果有函数lim f(x)=A，函数lim g(x)=B。

则有：

① lim[ f(x) + g(x) ] = lim f(x) + lim g(x) = A + B

② lim[ f(x) - g(x) ] = lim f(x) - lim g(x) = A - B

③ lim[ f(x) g(x) ] = lim f(x) lim g(x) = A * B

如果 g(x) 不等于0 ，则 lim[ f(x) / g(x) ] = lim f(x) / lim g(x) = A / B

Definition

设函数f(x)的定义域为D，对于任意的x∈D，均能找到一个大于0的M使得|f(x)|<=M，就称函数有界，如果这样的M不存在，就称函数无界

证明：

证明思路：

  1. 根据函数有界的定义，对于任意的M>0，均有|f(x)|<=M，只需找到一个M使得不等式不成立即可

  2. 因为x是变化的，所以我们要找到一个带x的表达式来表示M，让不等式成立或不成立，才能说明对于定义域内的任意的x，M都大于(即无界)或小于等于(即有界)函数值。

  3. 如果我们找到一个M不是用x来表示的而是一个固定的数值如一个很小的数值如0.0000001（或者很大的数值如1000000），那么我们是不能100%确定不等式永远成立的。

    (比如现在我们要证明函数无界即期望|f(x)|>M总是成立，根据直觉我们要找一个很大的M来证明不管M多么大不等式均成立，不妨设M为一千万，那么根据M的设定，我只需要让x=0.1(或者任意的零点几)，就能得出 即|f(x)|>M不成立了，所以我们无法证明或者说无法100%证明函数无界。这是为什么要用x来表示M的原因)

  4. 现在我们用x来表示M，因为函数有界的定义|f(x)|<=M，，在这道题中是 ｜1/x｜<= M，那让M = x 不行？或者等于x+1行不行
