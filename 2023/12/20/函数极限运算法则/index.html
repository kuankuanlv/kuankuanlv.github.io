无穷小相关的定理及定义

定义

有函数f(x)，当函数极限为0时，f(x)成为0趋于x0（或者0小于N）时候的无穷小。

  1. 无穷小是一个函数，非数值
  2. 0是一个特殊的无穷小

定理：与函数极限的关系

在自变量的同一变化过程x趋于x0或者x趋于无穷时，函数f(x)具有极限的充分必要条件是f(x)=A+å。

其中A是函数极限，å是无穷小

  1. 前提条件：f(x)=A+å，其中A是一个常数、å是一个无穷小
  2. 结论：那么其中的f(x)极限一定存在且为A

定理1：两个无穷小的和/差是无穷小

证明：

前提：有无穷小a和b，都是x趋于x0时候的无穷小。

设：

有 C = a + b

因a和b为无穷小，则有对于任意的ε>0,都存在δ>0，当0<|x-x0|<δ1时，满足|a|<ε；当0<|x-x0|<δ2时，满足|b|<ε；

我们让δ = min{δ1，δ2}，则δ是能同时满足a和b的一个δ，将a和b相加得到|a|+|b| = ε+ε①。

现我们对C求极限：lim C = lim (a + b)，即对于任意的ε>0，是否能找到一个δ>0，当0<|x-x0|<δ时，|C| <ε？

可以的，因为 |C| = ｜a+b｜，而且因为｜a+b｜<= |a| + |b|，又因为|a|+|b| = ε+ε①，所以｜a+b｜<2ε，(ε和常数相乘对结果无影响)，所以C是无穷小

函数极限的运算法则

基于上面无穷小的相关定理，可以简化函数极限的运算

定理：函数的极限的四则运算法则

如果有函数lim f(x)=A，函数lim g(x)=B。

则有：

① lim[ f(x) + g(x) ] = lim f(x) + lim g(x) = A + B

② lim[ f(x) - g(x) ] = lim f(x) - lim g(x) = A - B

③ lim[ f(x) * g(x) ] = lim f(x) * lim g(x) = A * B

如果 g(x) 不等于0 ，则 lim[ f(x) / g(x) ] = lim f(x) / lim g(x) = A / B
